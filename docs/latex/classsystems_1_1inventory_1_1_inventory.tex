\begin{DoxyVerb}Checks if Item is in Inventory.
        
        Key can be:
        str: returns True if str matches any ids in list of items
        dict {'id': str, 'count': int}: if str is in list of items, returns True if count is greater than Item.count, else False
        tuple (str, int): if str in list, returns int >= item.count, else False
        list [str, int]: same as tuple
        list [[str, int]]: for every list, same as tuple. Returns whether all are True.\end{DoxyVerb}
